기준 자료 역할
1) 정리된 분석자료 일부.pdf

이 파일은 최종 구조의 출발점으로 쓰는 게 맞습니다. 이유는 이 자료가 실제 업무 화면의 반복 구조를 보여주기 때문입니다. 이미 확인된 핵심은 이 UI가 업무분야 / 업무명 / 세부코드 / 개요 / 양식서류 / 첨부서류의 6필드 반복 구조라는 점입니다. 즉, 이 파일은 “무엇을 어떤 칸으로 나눌지”를 정하는 기준입니다.

여기서 중요한 건:

양식서류 = 작성서류(form_docs)
첨부서류 = 필요서류(supporting_docs)

이 둘은 절대 합치면 안 됩니다. 기존 정리에서도 이 부분이 시스템 이식의 핵심 전제로 잡혀 있습니다.

2) 221230 체류관리편람.pdf

이 파일은 주된 실질 근거로 쓰는 게 맞습니다. 편람 자체가 체류관리 일반사항, 체류자격별 관리기준, 그리고 체류자격외 활동 / 근무처 변경·추가 / 체류자격 부여 / 체류자격 변경허가 / 체류기간 연장 / 재입국허가 / 외국인 등록 등으로 체계화돼 있어, 세세분류별 데이터를 만들 때 가장 좋은 기준축입니다.

또한 편람은 제출서류 일반원칙도 명확히 잡고 있습니다. 예를 들어:

이미 제출해 보관 중인 서류는 생략 가능
불필요한 분량 많은 서류는 발췌 제출
심사를 위해 제출서류를 가감할 수 있음
이건 나중에 예외/공통 규칙 필드로 반드시 들어가야 합니다.
3) 260310 체류민원 자격별 안내 매뉴얼.pdf

이 파일은 최신 운영형 보정 자료로 쓰는 게 맞습니다. 목차 구조가 체류자격별 안내 + 공통사항으로 잘 정리돼 있고, 최근 실무에 맞는 항목이 반영돼 있습니다. 다만 이 파일은 최신 운영 보정용이고, 깊이와 세부 판단 기준은 여전히 221230 편람이 더 강합니다.

4) 나머지 자료
법령/시행령/시행규칙: 최종 법적 근거

white_font = Font(color="FFFFFF", bold=True)
bold = Font(bold=True)
thin = Side(style="thin", color="BFBFBF")
border = Border(left=thin, right=thin, top=thin, bottom=thin)
center = Alignment(horizontal="center", vertical="center", wrap_text=True)
wrap = Alignment(vertical="top", wrap_text=True)

def style_header(ws, row_idx):
    for cell in ws[row_idx]:
        if cell.value is not None:
            cell.fill = header_fill
            cell.font = white_font
            cell.border = border
            cell.alignment = center

def style_table(ws, start_row, end_row, cols):
    for r in range(start_row, end_row + 1):
        for c in range(1, cols + 1):
            cell = ws.cell(r, c)
            cell.border = border
            cell.alignment = wrap
    ws.freeze_panes = "A2"
