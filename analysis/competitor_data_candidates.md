## 5. SMS 라이브러리 파라미터 (sms.dll)

### 5-1. 메시지 발송 관련 파라미터
| 파라미터 | 추정 의미 |
|---|---|
| message | 메시지 본문 |
| templateId | 카카오 알림톡 템플릿 ID |
| groupId | 수신 그룹 ID |
| imageId | 첨부 이미지 ID |
| protocol | 발송 채널 (SMS/LMS/MMS/알림톡) |
| method | HTTP 메서드 |

### 5-2. 카카오 버튼 관련 파라미터
| 파라미터 | 추정 의미 |
|---|---|
| buttonName | 버튼 레이블 |
| buttonType | 버튼 유형 (링크/앱실행 등) |
| linkPc | PC 링크 URL |
| linkAnd | 안드로이드 링크 |

### 5-3. sms.dll에서 발견된 샘플 텍스트 (라이브러리 기본값)
- "안녕하세요."
- "홍길동님 회원가입을 환영합니다."
- "아래 '시작하기' 버튼을 통해 간단하게 사용방법을 익히실 수 있습니다."
- "현재 잔액:" / "현재 포인트:"
- "조회 결과"
- "전송 결과"
