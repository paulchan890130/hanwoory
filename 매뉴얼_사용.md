# K.ID 출입국업무관리 — 초보자용 사용 매뉴얼

> 이 문서는 프로그램을 처음 쓰는 직원이 실제 업무 화면을 따라 사용할 수 있도록 작성되었습니다.

---

## 1. 로그인

1. 브라우저에서 `http://localhost:3000` 주소 입력 후 Enter
2. 로그인 화면이 나타납니다
3. **아이디**와 **비밀번호** 입력 → **로그인** 버튼 클릭
4. 대시보드(홈 화면)가 열리면 로그인 성공

> 처음 접속할 때 화면 로드가 5~10초 걸릴 수 있습니다. 하얀 화면이 잠깐 보이는 것은 정상입니다.

---

## 2. 화면 구성

왼쪽에 세로 메뉴바가 있고, 오른쪽이 본문 영역입니다.

| 메뉴 이름 | 하는 일 |
|---|---|
| 대시보드 | 오늘 일정, 만기 예정 고객 현황 |
| 고객관리 | 고객 목록 조회, 등록, 수정 |
| OCR 스캔 | 여권·외국인등록증 사진 찍어서 고객 정보 자동 입력 |
| 업무관리 | 예정/진행/완료 업무 목록 관리 |
| 일정 | 달력 형태 일정 확인 및 등록 |
| 문서자동작성 | 비자 종류에 맞는 서류 묶음 PDF 자동 생성 |
| 통합검색 | 고객명·업무명 전체 검색 |
| 메모 | 단기·중기·장기 메모 관리 |
| 게시판 | 팀 공지 및 업무 노트 |
| 업무참고 | 비자별 업무 기준 정보 |
| 관리자 | 계정 생성·관리 (관리자만 보임) |

---

## 3. 고객관리 — 어떻게 쓰는가

### 3-1. 고객 목록 보기

왼쪽 메뉴 → **고객관리** 클릭
고객 목록 표가 나타납니다. 스크롤하거나 검색해서 원하는 고객을 찾습니다.

### 3-2. 고객 등록 (처음 등록하는 경우)

1. 고객관리 화면 → 우상단 **새 고객 추가** 버튼 클릭
2. 입력 양식이 열립니다

입력 항목:
| 필드 | 설명 |
|---|---|
| 한글 | 고객 한글 이름 (성+이름 합쳐서) |
| 성 | 영문 성 (여권의 Surname) |
| 명 | 영문 이름 (여권의 Given names) |
| 여권 | 여권 번호 |
| 국적 | 국적 코드 또는 국가명 |
| 성별 | M / F |
| 등록증 | 외국인등록증 번호 앞 6자리 |
| 번호 | 외국인등록증 번호 뒤 7자리 |
| 만기 | 여권 만기일 (YYYYMMDD 형식 또는 YYYY-MM-DD) |
| 만기일 | 등록증 만기일 |
| 연 / 락 / 처 | 전화번호 세 부분 (예: 010 / 1234 / 5678) |
| 체류자격 | 비자 종류 (F-1, F-4 등) |

3. 입력 완료 후 **저장** 클릭
4. 목록에 새 고객이 추가됩니다

### 3-3. 고객 정보 수정

1. 목록에서 해당 고객 행 클릭 → 상세 화면 열림
2. 수정할 필드 클릭 → 값 변경
3. **저장** 클릭 → Google Sheets에 즉시 반영

### 3-4. 만기 예정 알림 확인

대시보드 화면에 들어가면 여권 또는 등록증 만기일이 90일 이내인 고객 목록이 자동으로 표시됩니다.

---

## 4. OCR 스캔 — 여권·등록증 자동 입력

OCR 스캔을 사용하면 여권이나 외국인등록증 사진을 업로드하기만 해도 고객 정보가 자동으로 채워집니다.

### 사용 순서

1. 왼쪽 메뉴 → **OCR 스캔** 클릭
2. 화면 위쪽에서 **여권** 또는 **외국인등록증** 탭 선택
3. **파일 선택** 또는 **이미지 업로드** 버튼 클릭
4. 여권/등록증 사진 파일 선택 (JPG, PNG, PDF 지원)
5. **인식 시작** 클릭
6. 인식 결과가 오른쪽 입력 양식에 자동으로 채워집니다

### 결과 확인 및 저장

- 자동 인식 결과를 눈으로 확인합니다
- 잘못 인식된 부분이 있으면 직접 수정합니다
- **고객 저장** 버튼 클릭
  - 같은 여권번호 또는 등록번호의 고객이 이미 있으면 → "기존 고객 업데이트" 메시지가 뜨고 정보가 갱신됩니다
  - 새 고객이면 → "신규 고객 추가" 메시지가 뜹니다

### OCR이 잘 안 될 때

- 사진이 밝고 선명해야 합니다
- 여권 전체가 화면에 들어와야 합니다 (잘린 부분이 없어야 함)
- 기울어진 사진은 정확도가 낮습니다
- 스캔 이미지(300dpi 이상)가 스마트폰 사진보다 정확합니다

## 5. 업무관리 — 예정/진행/완료 업무

### 5-1. 업무 화면 구성

왼쪽 메뉴 → **업무관리** 클릭
화면은 세 탭으로 나뉩니다:

| 탭 | 내용 |
|---|---|
| 예정업무 | 아직 시작하지 않은 업무 |
| 진행업무 | 현재 처리 중인 업무 |
| 완료업무 | 처리가 끝난 업무 |

### 5-2. 새 업무 등록

1. **예정업무** 탭 → 우상단 **+ 업무 추가** 클릭
2. 입력:
   - 고객명 (연결할 고객 선택)
   - 업무 제목 (예: "F-4 재외동포 연장")
   - 마감일
   - 담당자
   - 메모
3. **저장** 클릭

### 5-3. 업무 상태 변경

- 예정 → 진행: 예정업무 행에서 **"진행 이동"** 버튼 클릭
- 진행 → 완료: 진행업무 행에서 **"완료 처리"** 버튼 클릭
- 완료업무는 조회만 가능하며 수정되지 않습니다

## 6. 문서자동작성 (quick-doc) — 서류 묶음 PDF 생성

비자 종류에 맞는 신청 서류를 한 번에 PDF로 묶어서 뽑아줍니다.

### 사용 순서

1. 왼쪽 메뉴 → **문서자동작성** 클릭
2. 다음 항목을 순서대로 선택합니다:
   - **카테고리**: 체류 / 귀화 / 영주 중 선택
   - **민원유형**: 등록 / 연장 / 자격변경 등
   - **비자종류**: F-1, F-2, F-4, H-2 등
3. 오른쪽에 "포함 서류 목록"이 표시됩니다
4. **고객 선택** (선택 사항): 고객을 선택하면 성명, 여권번호 등이 자동으로 채워집니다
5. **생성** 버튼 클릭
6. PDF 파일이 자동으로 다운로드됩니다

### 생성되는 파일 내용

- 해당 비자 종류에 필요한 서류들이 하나의 PDF 파일로 합쳐집니다
- 고객을 선택했다면 각 서류에 성명, 여권번호 등이 인쇄됩니다
- 파일명 예시: `F4_등록_홍길동_20240320.pdf`

### 오류 메시지가 뜰 때

"누락된 템플릿" 메시지가 뜨면 → 해당 서류 양식 파일이 `templates/` 폴더에 없는 것입니다. 운영자에게 파일 추가를 요청하세요.

---

## 7. 통합검색 — 전체 데이터 검색

1. 왼쪽 메뉴 → **통합검색** 클릭 또는 상단 검색창 사용
2. 검색어 입력 후 Enter
3. 고객, 업무, 메모, 게시판, 업무참고 전체에서 검색 결과를 한 화면에 보여줍니다

검색 팁:
- 고객 이름, 여권번호, 업무 제목으로 검색 가능
- 띄어쓰기로 여러 단어 입력하면 모두 포함된 결과만 표시 (AND 검색)
- 예: `홍길동 F-4` 입력 시 홍길동 관련 F-4 업무/고객만 표시

---

## 8. 메모

1. 왼쪽 메뉴 → **메모** 클릭
2. 탭 선택:
   - **단기메모**: 오늘~이번 주 처리할 사항
   - **중기메모**: 이번 달 안에 처리할 사항
   - **장기메모**: 중장기 계획, 특이사항 등
3. **+ 메모 추가** 클릭 → 내용 입력 → 저장
4. 저장된 메모는 목록에 표시되며 수정/삭제 가능

---

## 9. 일정

1. 왼쪽 메뉴 → **일정** 클릭
2. 달력 형태로 일정이 표시됩니다
3. 특정 날짜 클릭 → **+ 일정 추가** 클릭
4. 제목, 날짜, 내용 입력 → 저장
5. 일정은 달력에 표시되며 클릭하면 상세 내용 확인

---

## 10. 게시판

1. 왼쪽 메뉴 → **게시판** 클릭
2. 목록에서 공지/업무 내용 확인
3. **+ 글쓰기** 클릭 → 제목 및 내용 입력 → 등록
4. 게시물 클릭 → 댓글 작성 가능

---

## 11. 업무참고 (기준 정보)

비자별 접수 서류 기준, 처리 절차 등 업무에 필요한 참고 정보를 보관합니다.

1. 왼쪽 메뉴 → **업무참고** 클릭
2. 왼쪽에서 분류(탭) 선택
3. 오른쪽에 내용 표시

---

## 12. 전체 사용 흐름 (처음 사용 시 따라하기)

처음 이 시스템을 사용할 때 아래 순서로 테스트해 보세요.

### 1단계: 고객 등록 테스트

1. 고객관리 → 새 고객 추가
2. 아래 정보 입력:
   - 한글: `김테스트`
   - 성: `KIM`
   - 명: `TEST`
   - 여권: `M12345678`
   - 국적: `중국`
   - 만기: `20280101`
3. 저장
4. 목록에 "김테스트" 가 보이면 성공

### 2단계: OCR 스캔 테스트

1. OCR 스캔 → 여권 탭
2. 여권 샘플 이미지 업로드 (스캔본 또는 사진)
3. 인식 버튼 클릭 → 결과 확인
4. 저장 → "신규 고객 추가" 또는 "기존 고객 업데이트" 메시지 확인

### 3단계: 문서자동작성 테스트

1. 문서자동작성 → 체류 → 등록 → F-1 선택
2. 포함 서류 목록 확인
3. 생성 버튼 클릭 → PDF 다운로드 확인
4. 다운로드된 PDF 열어서 서류들이 합쳐져 있는지 확인

### 4단계: 통합검색 테스트

1. 통합검색 → "김테스트" 입력 → Enter
2. 고객 섹션에 "김테스트" 결과 표시 확인

### 5단계: 메모/일정 테스트

1. 메모 → 단기메모 → + 추가 → "테스트 메모" 저장
2. 일정 → 오늘 날짜 클릭 → + 추가 → "테스트 일정" 저장

여기까지 전부 됐으면 기본 기능이 정상 동작하는 것입니다.
