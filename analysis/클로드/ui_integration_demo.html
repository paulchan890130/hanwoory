1
데이터베이스 파일 외장하드 이전 준비 완료
현재 완성된 실무지침 데이터베이스 파일(정리.xlsx + immigration_guidelines_db_v2.json)을 외장하드로 복사합니다. 서버 프로그램도 같은 위치에 배치합니다.
2
공유기 포트 설정 설정 필요
공유기 관리 화면에서 포트 포워딩을 설정합니다.
외부 포트 8000 → 서버 컴퓨터 내부 IP의 8000번 포트로 연결합니다.
내부 네트워크(같은 와이파이)에서는 별도 설정 없이 바로 접속 가능합니다.
3
서버 프로그램 실행 설정 필요
외장하드가 연결된 컴퓨터(또는 공유기 자체 서버 기능)에서 서버 프로그램을 실행합니다.
컴퓨터 시작 시 자동 실행으로 설정하면 항상 켜진 상태로 유지됩니다.
실행 명령: uvicorn immigration_api:app --host 0.0.0.0 --port 8000
4
기존 시스템 접속 주소 변경 설정 필요
기존 Node.js 업무관리 시스템에서 구글 드라이브 주소를 내부 서버 주소로 변경합니다.
내부망 접속: http://192.168.x.x:8000 (서버 컴퓨터 내부 IP)
외부망 접속: 공유기 DDNS 주소 또는 고정 IP 사용
5
구글 드라이브 연동 코드 제거 순차 진행
기존 시스템에서 구글 드라이브 API 인증 코드, 토큰, 드라이브 파일 읽기 로직을 제거합니다.
데이터 조회는 전부 로컬 서버의 데이터베이스에서 직접 읽습니다.
인터넷 연결 없이도 내부망에서 정상 작동합니다.
6
데이터 업데이트 방식 운영 중 진행
하이코리아 메뉴얼 개정 시 정리.xlsx를 수정 후 서버를 재시작하면 즉시 반영됩니다.
구글 드라이브 동기화 불필요 — 외장하드 파일 직접 수정으로 운영합니다.
