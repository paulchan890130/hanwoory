# 일반귀화 준비서류

### 1. 본인 증명서류
여권（护照）, 영주증（永住证）, 증명사진 (흰판), 신분증（身份证）
호구부 (户口本) * 귀화추천서 2부 (국적자)

### 2. 중국 발급 서류
무범죄증명 + 공증 + 아포스티유 / 无犯罪记录 + 公证 + 海牙认证
친속 + 공증 + 아포스티유 / 亲属 + 公证 + 海牙认证

### 3. 재산 증명 서류 (6,000만원 이상 or 전년도 GNI 이상)
소득금액증명원 + 세부내역 (일용근로소득내역 / 원천징수영수증)
부동산 등기사항증명 / 예금통장 / 은행잔고증명 (6개월 이상)

### 4. 거주지 증명서류
집계약서 (租借合同书), 계약자 등록증 (合同人登陆证)
- 계약서 上 날짜 지난 경우 : 가스 / 전기 고지서 (煤气 / 电气 通知书)
