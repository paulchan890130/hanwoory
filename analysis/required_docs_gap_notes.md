## 7. 이름 불일치 주의 (동일 서식, 다른 이름)

경쟁사와 우리 앱이 동일한 서식을 다른 이름으로 부르는 케이스. PDF 파일명 정규화 시 참고.

| 경쟁사 이름 | 우리 앱 이름 |
|---|---|
| 숙소제공확인서 | 거주숙소 제공 확인서 |
| 외국인 직업 신고서 | 직업신고서 |
| 외국인 재학여부 신고서 / 재학사항 신고서 | 재학신고서 |
| 한글병기 신청서 | 한글성명 병기 신청서 |
| 대행업무수행확인서(외국인등록증 등 수령) / 업무수행확인서 | 대행업무수행확인서 |
| 비취업서약서(F-1) | 비취업 서약서 |
| 국내단순노무업종 비취업 서약서 | 단순노무 비취업 서약서 |
