프로그램의 대쉬보드에
'업무미리보기'버튼이 있고 이를 클릭하면 창이떠
여기서 정리된 방식은
업무종류 - 업무명 - 업무코드 를 선택하게 되어있고
업무종류를 클릭하면 선택으로
체류자격외활동, 자류자격부여, 체류자격변경, 체류기간연장, 재입국허가, 외국인등록, 기타업무, 사증발급인정, 근무처변경, 국적, 각종식청/청구 을 선택하게 나오고 여기서 하나를 선택하면 그다음 업무명에 업무종류에 맞는 업무명이 나열되고 선택 할 수 있게 되어있어
그리고 업무명에 나열된 항목을 클릭하면 업무코드에 상위 업무명에 맞는 항목들이 나열되어 선택하게 되어있어
이렇게 3가지를 탑-다운으로 선택하면
선택된 업무분야, 업무명, 세부코드가 표시되고
개요, 양식서류, 첨부서류가 세부적으로 선택되는 로직이야
내가 원하는 것은 하나하나 원하는 것을 클릭해서 보기 힘드니
프로그램에 세팅된 로직을 분석해서 한눈에 볼 수 있게
모든 조건별 개요, 양식서류, 첨부서류를 추출하라는 말이야
아니면 트리형식으로 해도 좋고

strings "C:\Users\윤찬\K.ID soft\analysis\2차 추출자료\iLinkData.bin" > C:\Users\윤찬\K.ID soft\analysis\2차 추출자료\menu_strings.txt

$binPath = "C:\Users\윤찬\K.ID soft\analysis\2차 추출자료\iLinkData.bin"          # ← 여기 경로를 실제로 복사한 경로로 바꾸세요
$outPath = "C:\Users\윤찬\K.ID soft\analysis\2차 추출자료\clean_menu.txt"

try {
    [byte[]]$bytes = [System.IO.File]::ReadAllBytes($binPath)
    $text = [System.Text.Encoding]::UTF16LE.GetString($bytes)

    $clean = $text -split "[\x00-\x1F\x7F-\x9F]+" | 
             Where-Object { $_.Trim().Length -gt 5 -and $_ -match '[가-힣a-zA-Z0-9]' }

    $clean | Out-File $outPath -Encoding utf8 -Force

    Write-Host "✅ 정리 완료! 파일 열기 →" $outPath -ForegroundColor Green
}
catch {
    Write-Host "❌ 오류: $($_.Exception.Message)" -ForegroundColor Red
}
