  출입국 실무지침 서버 — 설치 및 전환 안내
══════════════════════════════════════════════════════════

[ 준비물 ]
  • Windows 컴퓨터 (공유기에 연결된 것, 항상 켜두는 컴퓨터)
  • 외장하드 (공유기 USB 포트 또는 해당 컴퓨터에 연결)
  • Python 3.10 이상 설치 (https://www.python.org)

──────────────────────────────────────────────────────────
1단계 │ 파일 준비
──────────────────────────────────────────────────────────
외장하드(또는 지정 폴더)에 아래 파일을 모두 복사합니다.

  📁 immigration_server/          ← 폴더 이름은 자유
      ├── immigration_api.py                (서버 프로그램)
      ├── immigration_guidelines_db_v2.json (데이터베이스)
      ├── 서버_시작.bat                     (서버 실행 파일)
      └── 자동시작_등록.bat                 (자동실행 등록)

──────────────────────────────────────────────────────────
2단계 │ 서버 시작 테스트
──────────────────────────────────────────────────────────
"서버_시작.bat" 를 더블클릭합니다.
검은 창이 뜨고 아래 메시지가 나오면 성공입니다.

  Uvicorn running on http://0.0.0.0:8000

같은 와이파이에 연결된 다른 컴퓨터에서 브라우저를 열고
http://서버IP:8000 으로 접속해서 확인합니다.
(서버 IP는 검은 창에 표시됩니다)

──────────────────────────────────────────────────────────
3단계 │ 컴퓨터 시작 시 자동 실행 등록
──────────────────────────────────────────────────────────
"자동시작_등록.bat" 를 오른쪽 클릭 → 관리자 권한으로 실행

이후 컴퓨터를 켜면 서버가 자동으로 시작됩니다.

──────────────────────────────────────────────────────────
4단계 │ 공유기 포트 포워딩 (사무실 외부에서도 쓸 경우)
──────────────────────────────────────────────────────────
공유기 관리 화면(보통 192.168.0.1 또는 192.168.1.1) 접속
→ 포트포워딩 또는 가상서버 메뉴
→ 외부포트 8000 → 서버컴퓨터 내부IP → 내부포트 8000 추가

※ 사무실 내부(같은 와이파이)에서만 쓴다면 이 단계 불필요

──────────────────────────────────────────────────────────
5단계 │ 기존 Node.js 시스템 연동 주소 변경
──────────────────────────────────────────────────────────
기존 시스템의 .env 파일(또는 설정 파일)에 아래 줄 추가:

  IMMIGRATION_API_URL=http://192.168.x.x:8000
  (x.x 부분에 서버 컴퓨터의 실제 내부 IP 입력)

기존 구글 드라이브 연동 모듈을 아래로 교체:
  기존: require('./구_구글드라이브모듈')
  변경: require('./immigration_client_local')

──────────────────────────────────────────────────────────
6단계 │ 데이터 업데이트 방법 (이후 운영)
──────────────────────────────────────────────────────────
하이코리아 메뉴얼 개정 등으로 데이터를 바꿔야 할 때:

  1. 정리.xlsx 수정
  2. Python 스크립트로 JSON 재생성:
     > python 엑셀_json_변환.py
  3. 서버 재시작 (창 닫고 서버_시작.bat 다시 실행)

구글 드라이브 동기화 불필요 — 파일 직접 수정으로 즉시 반영

──────────────────────────────────────────────────────────
[ 문제 해결 ]
──────────────────────────────────────────────────────────
● 서버가 시작되지 않음
  → Python이 설치되어 있는지 확인 (python --version)
  → pip install fastapi uvicorn 수동 실행

● 다른 컴퓨터에서 접속 안됨
  → 서버 컴퓨터의 Windows 방화벽에서 8000번 포트 허용
  → 제어판 → Windows Defender 방화벽 → 고급 설정
    → 인바운드 규칙 → 새 규칙 → 포트 → 8000 허용

● 기존 구글 드라이브 연동이 계속 사용됨
  → Node.js 코드에서 require 경로를 새 파일로 교체했는지 확인
  → .env 파일의 IMMIGRATION_API_URL 설정 확인
