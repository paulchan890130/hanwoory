# 01_F2_동포_수정메모

## 상태
이번 산출물은 **확정본이 아니라 검수초안**입니다.

이유:
- 현재 `/mnt/data` 기준으로 **체류민원 자격별 안내 매뉴얼 원문 PDF가 없음**
- 따라서 이번 검수는 업로드된 `업무정리_정밀보정_F2_동포.xlsx`의 구조와 코드 분해 상태를 점검하고,
  누락 가능성이 있는 구간을 표시하는 수준까지 진행함

## 이번에 한 작업
1. 기존 `업무정리_정밀보정_F2_동포.xlsx`를 기준으로 01단계 범위를 재점검
2. 새 시트 `01_검수현황` 추가
3. 아래 항목들의 반영 여부와 절차 분리 상태를 표로 정리
   - F-2-2 / F-2-3 / F-2-4 / F-2-5 / F-2-7 / F-2-71 / F-2-99
   - C-3-8 / H-2 / F-3 / F-1
   - F-4 세부대상 / F-4 체류관리 / F-5 동포영주
4. 원문 PDF 없이 확정하기 어려운 구간은 `원문 PDF 대조 필요` 또는 `원문 확인 필요`로 표시

## 현재 확인된 점
- `F-2-01` 같은 통합코드 뭉개기는 현재 워크북에서 보이지 않음
- F-2는 최소한 다음 코드들로 분해되어 있음
  - F-2-2
  - F-2-3
  - F-2-4
  - F-2-5
  - F-2-7
  - F-2-71
  - F-2-99
- 동포 파트도 `관리` 한 줄 처리 대신
  - C-3-8
  - H-2
  - F-3/F-1
  - F-4 세부대상
  - F-4 체류관리
  - F-5
  로 나뉘어 있음

## 현재 보류한 점
- F-2-7은 현재 워크북상 `체류기간연장허가`만 뚜렷하게 보임
  - 변경 / 부여가 원문에 따로 있는지 원문 PDF 대조 필요
- C-3-8은 현재 워크북상 `사증발급` 위주로 보임
  - 연장 / 변경 / 부여 등 별도 절차 존재 여부 원문 PDF 대조 필요
- F-4 세부대상과 F-4 체류관리는 분리되어 있으나
  - 각 세부유형별 실제 제출서류 문구가 원문과 1:1 일치하는지는 원문 PDF 대조 필요

## 다음 바로 할 작업

2. 누락 소제목 표시
3. 절차구분 누락 보완
4. `01_F2_동포_확정본.xlsx`로 재산출
